# UltRenderer
本项目是一个代码不依赖于任何第三方库的软渲染器，其中实现了一套模仿OpenGL ES 2.0的软光栅管线，与一套路径追踪渲染器。

### 光栅化
### 路径追踪

## 依赖
由于代码不依赖于任何第三方库，因此只对编译环境有依赖:
- CMake 3.26（最低版本没有经过严格测试）
- C++20
